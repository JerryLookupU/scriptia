# 交互清单（页面按钮与交互说明）

## 主聊天页（中间 Chat + 右侧双区）

页面位置：顶部导航
位置名称：账户区
区域描述：登录/用户设置/语言/配色
按钮说明：登录、用户设置、语言切换、配色切换
交互说明：点击展开下拉面板；设置实时生效并记忆

页面位置：中间主区
位置名称：对话区（Chat）
区域描述：核心对话与卡片输出区
按钮说明：历史记录、停止生成、清空对话、复制结果
交互说明：聊天内容可滚动；聊天框固定在页面最下方

页面位置：中间主区
位置名称：历史记录
区域描述：会话缩略列表
按钮说明：会话条目
交互说明：点击“历史记录”弹出缩略会话列表；选择后跳转到该会话

页面位置：中间主区
位置名称：输入框
区域描述：指令输入与卡片引用入口
按钮说明：发送、引用卡片、随机生成卡片
交互说明：输入自然语言；引用卡片以引用标签进入输入框；点击“随机生成卡片”后在对话中弹出卡片

页面位置：中间主区
位置名称：消息卡片（对话输出）
区域描述：AI 生成的结果卡片
按钮说明：引用、查看、折叠、生成配图、保存
交互说明：引用插入上下文；查看以弹窗展示；折叠为单行样式；生成配图弹出独立窗口；保存进入右侧暂存区

页面位置：右侧上方
位置名称：会话暂存区
区域描述：当前对话内临时卡片集合
按钮说明：保存到卡片库、移除
交互说明：将暂存区卡片保存到卡片库（按类型归档）；可从暂存区移除

页面位置：右侧下方
位置名称：卡片库
区域描述：卡片类型浏览与管理
按钮说明：类型切换、搜索、筛选、收藏、引用
交互说明：按类型浏览卡片；点击卡片可引用到输入框或查看详情

页面位置：右侧下方
位置名称：卡片详情
区域描述：卡片字段与元信息
按钮说明：查看、复制、收藏
交互说明：全局卡片只读；复制后成为项目卡片并可编辑

## 卡片四种图式

页面位置：任意
位置名称：折叠样式
区域描述：表格单行样式
按钮说明：展开
交互说明：点击展开切换为对话展示样式或弹窗

页面位置：任意
位置名称：弹出窗口样式
区域描述：卡片详情弹窗
按钮说明：关闭、复制、生成配图
交互说明：关闭返回原页面；可生成配图作为卡片配图

页面位置：任意
位置名称：输入框引用样式
区域描述：引用标签
按钮说明：移除
交互说明：引用标签作为 message item 插入上下文；可一键移除

页面位置：任意
位置名称：对话框展示样式
区域描述：消息卡
按钮说明：引用、查看、折叠、生成配图
交互说明：消息卡支持操作并可进入暂存区
